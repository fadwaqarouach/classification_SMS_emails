
# Data 
This dataset contains English-language SMS messages that have been categorized as either "ham" (legitimate) or "spam." Each entry consists of a single message labeled accordingly as spam or ham.


# Workflow stages 
- Data understanding and exploration
- Feature engineering
- Exploratory data analysis (EDA)
- Data Preprocessing
- Model building and evaluation
- creating the web app with streamlit 
- App deployment with Git and Heroku  

# Requirements
- Python 3
- Pip (To Install packages)
- Streamlit
- Pandas
- Numpy
- Sickitlearn
- nltk

# Run Application on localhost :
```python
streamlit run app/app.py
```
